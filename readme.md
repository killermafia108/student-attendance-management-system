# Student Attendance Management System

## Project Overview

The Student Attendance Management System is a simple web-based application developed to manage student attendance records. This project allows users to mark student attendance and view attendance reports through an easy-to-use interface.

## Features

* Student Login Page
* Add Student Attendance
* View Attendance Records
* Attendance Report Page
* Simple and User-Friendly Interface

## Technologies Used

* HTML
* CSS
* JavaScript
* PHP
* SQL

## Project Files

* login.html
